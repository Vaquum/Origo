# How to add new assets?

1) Add the necessary asset file/s following the current naming convention
2) Make changes in `origo_control_plane/definitions.py`
3) Follow the instructions at the head of `origo_control_plane/definitions.py`
4) Make Pull Request with the changes committed

NOTE: For the changes to take effect in `origo`, user `root` must run the command `@deploy` on the server.
